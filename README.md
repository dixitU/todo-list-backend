# todo-list-backend

This is an backend project of todo-list app created with Node JS/Express server.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

   or

   ```bash
   yarn
   ```

2. Start the app

   ```bash
    npm run start
   ```
